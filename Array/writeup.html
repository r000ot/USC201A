Dynamic Array

Background:

This lab will require you to create a class that extends the Java array functionality.

Assignment:

You are to create a DynamicArray class which has a private data as an array of objects. See the DynamicArrayInterface for the methods to implement.

Requirements:

You should use the System.arraycopy method for copying arrays.

You should not use Java'sArrayList class.

All methods must be non-static.
